Portfólio

Aqui estão alguns dos meus projetos:

  • Teclado Consciente TIM

    O Teclado Consciente TIM é um aplicativo de teclado desenvolvido para promover a sustentabilidade e a consciência ambiental entre seus usuários. O app oferece sugestões de palavras e frases que incentivam práticas sustentáveis, além de fornecer informações úteis sobre o impacto ambiental das escolhas diárias. Participei do desenvolvimento completo do aplicativo, focando em arquitetura MVVM, integração com API REST e implementação de funcionalidades personalizadas.

  • PokeListCompose

    O PokeListCompose é um aplicativo desenvolvido utilizando Jetpack Compose para exibir uma lista de Pokémons, consumindo uma API REST. O projeto demonstra o uso de padrões modernos de desenvolvimento, como a arquitetura MVVM, e integrações com coroutines para gerenciar chamadas assíncronas. O app também é um exemplo prático de como utilizar a nova abordagem declarativa do Jetpack Compose para construir interfaces de usuário modernas e reativas no Android.

    Visite minha página de desenvolvedor na Google Play Store para ver uma coleção de aplicativos Android que desenvolvi e publiquei.
